# SPD矩阵流形基础

# SPD矩阵

对称正定矩阵定义为

Sn++={P∈Rn×n∣uTPu≻0,∀u∈Rn}\mathcal S_n^{++} = \{P\in\mathbb R^{n\times n}| \boldsymbol{u}^TP \boldsymbol u \succ 0, \forall \boldsymbol u \in \mathbb R^n\} Sn++​={P∈Rn×n∣uTPu≻0,∀u∈Rn}

对矩阵P∈Sn++P\in \mathcal S_n^{++}P∈Sn++​进行特征分解

P=Udiag(σ1,…,σn)UTP=U \text{diag}(\sigma_1,\ldots,\sigma_n)U^T P=Udiag(σ1​,…,σn​)UT

则矩阵指数算子和对数算子定义如下

exp⁡(P)=Udiag(exp⁡(σ1),…,exp⁡(σn))UTlog⁡(P)=Udiag(log⁡(σ1),…,log⁡(σn))UT\begin{aligned} \exp(P) &= U \text{diag}(\exp(\sigma_1),\ldots,\exp(\sigma_n))U^T\\ \log(P) &= U \text{diag}(\log(\sigma_1),\ldots,\log(\sigma_n))U^T \end{aligned} exp(P)log(P)​=Udiag(exp(σ1​),…,exp(σn​))UT=Udiag(log(σ1​),…,log(σn​))UT​

# 黎曼度量

Sn++\mathcal S_n^{++}Sn++​是一个可微的黎曼流形，任意点P∈Sn++P\in \mathcal S_n^{++}P∈Sn++​的导数构成的切空间TPT_PTP​是一个对称矩阵空间。

在切空间任取两点S1,S2∈TpS_1,S_2\in T_pS1​,S2​∈Tp​，可选择局部内积来定义SPD流形Sn++\mathcal S_n^{++}Sn++​上的度量(natural metric)

⟨S1,S2⟩P=tr(S1P−1S2P−1)\left\langle S_1, S_2\right\rangle_P = \text{tr}(S_1P^{-1}S_2P^{-1}) ⟨S1​,S2​⟩P​=tr(S1​P−1S2​P−1)

该内积自然诱导出切空间的范数

∥S∥P2=⟨S,S⟩P=tr(SP−1SP−1)\|S\|_P^2 = \left\langle S, S\right\rangle_P = \text{tr}(SP^{-1}SP^{-1}) ∥S∥P2​=⟨S,S⟩P​=tr(SP−1SP−1)

当P=IP=IP=I时，该范数等价于Frobenius范数，即∥S∥F2=⟨S,S⟩I\|S\|_F^2 = \left\langle S, S\right\rangle_I∥S∥F2​=⟨S,S⟩I​。

# 黎曼距离

设流形上曲线Γ(t):[0,1]→Sn++\Gamma(t):[0,1]\to\mathcal S_n^{++}Γ(t):[0,1]→Sn++​的起点Γ(0)=P1\Gamma(0)=P_1Γ(0)=P1​和终点Γ(1)=P2\Gamma(1)=P_2Γ(1)=P2​，则曲线的长度可表示为积分形式

L(Γ(t))=∫01∥Γ˙(t)∥Γ(t)dtL(\Gamma(t))=\int_0^1\|\dot\Gamma(t)\|_{\Gamma(t)} dt L(Γ(t))=∫01​∥Γ˙(t)∥Γ(t)​dt

其中Γ˙(t)∈TΓ(t)\dot\Gamma(t)\in T_{\Gamma(t)}Γ˙(t)∈TΓ(t)​为切向量，∥⋅∥⋅\|\cdot\|_{\cdot}∥⋅∥⋅​为切空间的范数。流形上连接P1P_1P1​和P2P_2P2​的最短长度曲线称为测地线。上述内积和范数可诱导出流形Sn++\mathcal S_n^{++}Sn++​上的测地线距离，即黎曼距离。

δR2(P1,P2)=∥log⁡(P1−1P2)∥F2=∑i=1nlog⁡2λi\delta_R^2 (P_1,P_2) = \|\log (P_1^{-1}P_2)\|_F^2 = \sum_{i=1}^n \log^2 \lambda_i δR2​(P1​,P2​)=∥log(P1−1​P2​)∥F2​=i=1∑n​log2λi​

其中λi\lambda_iλi​是P1−1P2P_1^{-1}P_2P1−1​P2​的第iii个实特征值。该距离具有如下特性：

  • δR(P1,P2)=δR(P2,P1)\delta_R (P_1,P_2) = \delta_R (P_2,P_1) δR​(P1​,P2​)=δR​(P2​,P1​)

  • δR(P1,P2)=δR(P1−1,P2−1)\delta_R (P_1,P_2) = \delta_R (P_1^{-1},P_2^{-1}) δR​(P1​,P2​)=δR​(P1−1​,P2−1​)

  • δR(P1,P2)=δR(WTP1W,WTP2W),∀W∈Gl(n)\delta_R (P_1,P_2) = \delta_R (W^TP_1W,W^TP_2W),\forall W\in \text{Gl(n)} δR​(P1​,P2​)=δR​(WTP1​W,WTP2​W),∀W∈Gl(n)

最后一个性质验证了仿射不变性(affine invariance)，因此该黎曼度量也称为仿射不变黎曼度量(Affine Invariant Riemannian Metric, AIRM)。

# 小结

上面介绍了流形上的分类算法，其中很大篇幅都是介绍流形上的操作，因此掌握了这些基础后可以尝试进行新的改进。不过鉴于文章是2012年发表的，后面已经有很多改进算法，能想到的和不能想到的估计都已经做遍了，所以简单的小改进是没多大意义的。这也反过来说明优秀的文章不仅仅需要找到合适的切入点，还需要符合时代的潮流。
