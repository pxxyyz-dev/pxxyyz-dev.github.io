# Lasso

在最小二乘的惩罚下添加对回归系数的ℓ1\ell_1ℓ1​范数的约束

β^lasso=arg⁡min⁡β∥Y−∑j=1pXjβj∥2+λ∑j=1p∣βj∣\hat\beta_{\text{lasso}} = \arg\min_\beta \|Y - \sum_{j=1}^p X_j \beta_j\|^2 + \lambda \sum_{j=1}^p |\beta_j| β^​lasso​=argβmin​∥Y−j=1∑p​Xj​βj​∥2+λj=1∑p​∣βj​∣

  • 可通过最小角度回归LARS算法高效求解。

  • 不适于样本少而维数高的变量选择，回归系数至多有nnn个非零项，这是因为凸优化问题的性质。

  • 除非系数的ℓ1\ell_1ℓ1​范数的边界小于某个值(由权衡参数λ\lambdaλ确定)，否则Lasso不是良定义的(well-defined)。

  • Lasso无法进行分组变量(grouped variables)选择，它倾向于从组中选择一个变量，而忽略其他变量。

# Elastic Net

为了利用所有变量的信息，对岭回归和lasso进行凸组合

β^en=(1+λ2)arg⁡min⁡β∥Y−∑j=1pXjβj∥2+λ2∑j=1p∣βj∣2+λ1∑j=1p∣βj∣\begin{aligned} \hat\beta_{\text{en}} = (1+\lambda_2)&\arg\min_\beta \|Y - \sum_{j=1}^p X_j \beta_j\|^2 \\ &+ \lambda_2 \sum_{j=1}^p |\beta_j|^2 + \lambda_1 \sum_{j=1}^p |\beta_j| \end{aligned} β^​en​=(1+λ2​)​argβmin​∥Y−j=1∑p​Xj​βj​∥2+λ2​j=1∑p​∣βj​∣2+λ1​j=1∑p​∣βj​∣​

  • LARS-EN算法高效求解。
  • 当λ2=0\lambda_2 = 0λ2​=0时，Elastic Net退化为Lasso。
  • 引入的二次项
    • 去除变量选择个数的限制，即适合样本少而维数高的情况
    • 有利于分组变量选择(grouping effect)
    • 稳定了ℓ1\ell_1ℓ1​正则化的路径

# Elastic Net 正则化的几何性质

J(β)=α∥β∥2+(1−α)∥β∥1≤tJ(\beta) = \alpha \|\beta\|^2 + (1 - \alpha)\|\beta\|_1 \leq t J(β)=α∥β∥2+(1−α)∥β∥1​≤t

  • 严格凸的边，凸度的强度随α\alphaα而变化（grouping）

∥M−NAT∥2=tr(MTM)−2tr(MTNAT)+tr(ANTNAT)∝−2tr(MTNAT)+tr(NTNATA)∝−2tr(MTNAT)=−2tr(UDVTAT)=−2tr(UDA∗T)=−2tr(A∗TUD)\begin{aligned} &\| M - N A^T\|^2 \\ & = \text{tr}(M^T M) - 2\text{tr}(M^T NA^T) + \text{tr}(AN^TNA^T) \\ &\propto- 2\text{tr}(M^T NA^T) + \text{tr}(N^TNA^TA)\\ &\propto -2\text{tr}(M^T NA^T)\\ &= -2\text{tr}(UDV^TA^T)\\ &= -2\text{tr}(UD{A^*}^T)\\ &= -2\text{tr}({A^*}^T UD)\\ \end{aligned} ​∥M−NAT∥2=tr(MTM)−2tr(MTNAT)+tr(ANTNAT)∝−2tr(MTNAT)+tr(NTNATA)∝−2tr(MTNAT)=−2tr(UDVTAT)=−2tr(UDA∗T)=−2tr(A∗TUD)​

其中A∗=AVA^* = AVA∗=AV满足A∗TA∗=I{A^*}^T A^* = IA∗TA∗=I，对角阵DDD由奇异值组成，因此优化问题寻找A∗TU{A^*}^T UA∗TU对角元素为最大正数的条件。利用Cauchy-Schwartz不等式可知，当A∗=UA^* = UA∗=U时取最优解。因此A^=UVT{\hat A} = U V^TA^=UVT。

# 算法流程

# 方差调整

传统PCA算法得到的主成分是无关的，载荷是相互正交的，但是稀疏化后无关性不在满足，这是因为Z^TZ^\mathrm {\hat Z}^T \mathrm {\hat Z}Z^TZ^不是严格的对角矩阵，为此需要对每个主成分做类似Schmidt 正交化，对每个主成分减去前面特征值方向的分量，从而保证正交性

Z^j⋅1,…,j−1=Z^j−H1,…,j−1Z^j\mathrm {\hat Z}_{j\cdot 1,\ldots,j-1} = \mathrm {\hat Z}_{j} - \mathrm H_{1,\ldots,j-1} \mathrm {\hat Z}_{j} Z^j⋅1,…,j−1​=Z^j​−H1,…,j−1​Z^j​

其中H1,…,j−1\mathrm H_{1,\ldots,j-1}H1,…,j−1​是投影矩阵，调整后主成分的方差为∑j=1k∥Z^j⋅1,…,j−1∥2=tr(Z^TZ^)\sum_{j=1}^k \|\mathrm {\hat Z}_{j\cdot 1,\ldots,j-1}\|^2 = \text{tr} (\mathrm {\hat Z}^T \mathrm {\hat Z})∑j=1k​∥Z^j⋅1,…,j−1​∥2=tr(Z^TZ^)（如果无关）。
